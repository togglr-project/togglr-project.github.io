Togglr

A powerful feature flag and experimentation platform that extends classic feature flags into a full experimentation framework with adaptive learning algorithms.

Get Started View on GitHub

Experimentation & Algorithms

Togglr extends classic feature flags into a full experimentation framework. Each feature can have multiple variants (A/B/N arms), and you can attach an algorithm that dynamically controls which variant is shown to users — optimizing for reward or conversion automatically.

ε

ε-Greedy

Explores randomly with probability ε, otherwise exploits the best variant for optimal performance.

T

Thompson Sampling

Bayesian approach that balances exploration and exploitation for intelligent decision making.

U

Upper Confidence Bound

Selects the variant with the highest confidence bound for expected reward optimization.

Key Features

Multi-Variant Experiments

Run A/B/N tests with multiple variants and let algorithms automatically optimize for the best performing option.

Adaptive Learning

Algorithms learn over time and adjust rollout dynamically — no manual tweaking required.

Real-time Analytics

Track statistics, successes, failures, and rewards per variant with comprehensive dashboards.

Guarded Features

Pending changes with approval workflow ensure safe deployments and controlled rollouts.

User Segmentation

Target specific user segments with sophisticated targeting rules and context-aware decisions.

Smart Scheduling

Automatic activation/deactivation based on schedules, time zones, and business rules.

Enterprise Security

LDAP, SSO/SAML (Keycloak), and two-factor authentication for enterprise-grade security.

Real-time Monitoring

Health monitoring, auto-disable on errors, and comprehensive audit logs for production safety.

Architecture

Built with modern technologies for scalability and reliability:

Backend

Go with TimescaleDB, NATS JetStream, REST API, and WebSocket broadcaster for real-time updates.

Frontend

React + TypeScript with Material-UI for a modern, responsive user interface.

SDKs

Multi-language support: Go, Python, Ruby, TypeScript, PHP, Rust and Elixir for seamless integration.
